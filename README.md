# Sudoku Game

## Introduction

An easy-to-play web-based Sudoku game

![Sudoku game image](./assets/image.png)

**[Play Now!](https://kangmoo.github.io/sudoku/)**

## Keyboard Shortcuts

- Number keys (1-9): Enter numbers
- Arrow keys: Move between cells
- `H`: Use hint
- `M`: Toggle note mode
- `Delete/Backspace`: Delete input
